# Reasoning: Grant Application Writer

This document captures the thinking behind the prompt. It exists so a reader can understand not just *what* the prompt is, but *why* it ended up this way.

## Goal

Write a professional 750-word grant application for the Buddhapadipa Temple Trust's Thai Language for All Sunday School Programme, applying for £50,000. The prompt was written for a real submission, not just an exercise.

## Iteration history

Single prompt, used once. The constraints were baked in upfront (professional tone, academic English, 750 words, critical thinking) rather than refined across multiple turns — suggesting deliberate prompt design rather than trial and error.

## Failure modes the final version handles

The constraints guard against two common AI writing failures — generic or informal tone ("write as a professional grant writer") and vague padding ("750 word exercise"). By specifying both the register and the length, the prompt steers away from outputs that read like a template rather than a real application.

## Outcome

The output was submitted, adapted, and used. It was also later used as the test case when building Grant Matcher Pro — the Buddhapadipa Temple Trust appears as the example organisation in the app's profile form placeholder.

## What I'd change next

Add the specific funder's name and criteria to the prompt so the language can be tailored to their priorities (e.g. "applying to the National Lottery Community Fund").
