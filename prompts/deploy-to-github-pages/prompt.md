# Deploy to GitHub Pages

> **Category:** agent-workflow | **Status:** production | **Last updated:** 2026-05-16

## The prompt

```
I need to deploy my Grant Matcher Pro application to GitHub and enable GitHub Pages. Here's what I need:

PROJECT DETAILS:
- Project Name: grant-matcher
- Description: AI-powered grant matching platform to help organizations find funding opportunities
- Files: index.html and README.md (already downloaded to my computer)

TASKS:
1. Create a new GitHub repository called "grant-matcher"
2. Initialize it with a proper .gitignore for web projects
3. Add the index.html file as the main application
4. Add the README.md file with project documentation
5. Create a LICENSE file (MIT License)
6. Set up GitHub Pages to serve from the main branch
7. Create an initial commit with message: "Initial commit: Grant Matcher Pro v1.0"
8. Enable GitHub Pages and confirm the live URL

ADDITIONAL SETUP:
- Add topics/tags to the repo: grant-matching, funding, nonprofit, charity, web-app, portfolio-project
- Make the repository public
- Add a nice repository description: "Smart grant matching platform helping organizations discover relevant funding opportunities"
```
